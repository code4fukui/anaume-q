# anaume-q

Set the OPENAI_API_KEY environment variable in the .env file.

```
OPENAI_API_KEY=xxxxx
```

Run the script to generate fill-in-the-blank questions:

```sh
deno run -A https://code4fukui.github.io/anaume-q/fetchQ.js "AIは万能である"
```
